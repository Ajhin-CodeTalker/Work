Can They Really Learn?

Game-Based Java Learning

Java Realms: The Java Chronicle System is an interactive game designed to teach students Java programming through gameplay. It offers hands-on learning by integrating coding concepts into an RPG environment, enabling users to build both programming skills and game logic.

Gameplay Mechanics

Players control their character using W, A, S, D or arrow keys. Interact or attack using T, and fire ranged projectiles with E, which consumes mana. Blue mana crystals indicate availability; white means you're out.

Puzzles & Progression System

To collect items like Lanterns or Keys, players must answer questions correctly, earning speed boosts for faster movement. Each key has a unique name and opens only its matching door. Using the wrong key triggers a hint to guide the player.
